# List 인터페이스

 List 인터페이스는 중복을 허용하면서 저장순서가 유지되는 컬렉션을 구현하고 Collection 인터페이스로 부터 메서드를 상속받았다.

|Object get(int index)|
|int indexOf(Object o)|
