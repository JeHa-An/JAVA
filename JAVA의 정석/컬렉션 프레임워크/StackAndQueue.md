# Stack과 Queue
&nbsp;**Stack**은 마지막에 저장한 데이터를 가장 먼저 꺼내는 **LIFO(Last In First Out)**구조이고, **Queue**은 처음 저장한 데이터를 가장 먼저 
꺼내는 *FIFO(First In First Out)*구조이다. <br>
