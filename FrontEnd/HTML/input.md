# input
 form 태그 내부에서 사용되며, 사용자가 정보를 입력하는 부분(입력 필드)을 만들어야 할 때 사용

## 속성
- name
>

- readonly 
> 읽기 전용 필드

- value
> 
- placeholder
> 힌트 표시

> 텍스트 필드에 최대로 입력할 수 있는 문자의 개수 지정

- required
> 필수 입력 필드 지정(입력 값이 없으면 안됨)

- type
> input 태그의 입력 방식 지정

|type 속성|설명|
|--|--|
|text|텍스트 입력|
|hidden|화면상에는 표시되지 않지만 서버에 입력 form을 전송시 함께 전송|
|email|이메일 입력|
|file|파일 첨부 버튼|
|password|비밀번호 입력|
|button|버튼 삽입|
|checkbox|체크박스 삽입|
|radio|라디오 버튼 삽입 |
|submit|서버 전송 버튼|

- width
- src
- size
- maxLength
- max, min
- checked
- autofocus
